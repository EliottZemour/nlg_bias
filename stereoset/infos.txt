In the preds/ folder, there are the probabilities predicted by the models. The file names are the same as the file names for the parent folder (i.e the results).

stereoset/dexperts_med_stereoset_with_anti.json
dexperts = DExperts(
    base_model='gpt2-medium',
    antiexpert_model='eliolio/gpt2-finetuned-redditbias',
    expert_model='eliolio/gpt2-finetuned-reddit-antibias',
    tokenizer='gpt2',
)
alpha = 2.0
